## Cakupan Sistem

- Website publik untuk pemasaran, katalog produk, dan kontak/inquiry.
- Dashboard admin untuk operasional bisnis, stok, penjualan, keuangan, procurement, dan CMS website.
- API internal dengan autentikasi Sanctum untuk kebutuhan integrasi.

## Catatan

Repositori ini adalah gambaran lengkap sistem yang ada, dan bukan berisi source code aplikasi. Dokumentasi ini bertujuan untuk memberikan panduan penggunaan dan operasional bagi pengguna yang telah membeli sistem.
