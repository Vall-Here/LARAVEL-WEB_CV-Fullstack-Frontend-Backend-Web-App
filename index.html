Fitur yang Ditawarkan

Website Publik

  • Beranda, Tentang, Katalog, Detail Produk, Kontak
  • Inquiry form dengan validasi, honeypot, dan rate limit
  • Konten dinamis dari CMS

Dashboard Admin

  • Produk, stok, CRM, penjualan, keuangan, procurement
  • Kasir Desk dan Financial Report Center
  • Kelola Website untuk update konten publik

Integrasi

  • WhatsApp quick action transaksi
  • Lampiran dokumen
  • Export PDF dan laporan periodik

Kontrol Akses

  • Role: super_admin, owner, admin, kasir, gudang
  • Permission granular per modul
  • Policy authorization dan activity log

Alur Kerja Sistem

  1. Pelanggan melihat produk di website dan kirim inquiry.
  2. Admin memproses inquiry menjadi quotation.
  3. Quotation dikonversi menjadi sales order.
  4. Order diproses, invoice diterbitkan, pembayaran dicatat.
  5. Data masuk ke laporan dan monitoring KPI.

Flow ringkas

Inquiry -> Quotation -> Sales Order -> Invoice -> Payment -> Report

Alur Per User

super_admin

Kelola konfigurasi, akses, data master, audit, dan intervensi lintas modul.

owner

Review KPI, cashflow, piutang, pengeluaran, dan keputusan strategis.

admin

Menjalankan alur inquiry sampai invoice dan update CMS website.

kasir

Mencatat pembayaran invoice, validasi nominal, dan monitor outstanding.

gudang

Kelola mutasi stok, penerimaan barang, dan cek stok kritis.
