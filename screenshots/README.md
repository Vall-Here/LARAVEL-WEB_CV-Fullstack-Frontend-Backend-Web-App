# Folder Screenshots

Folder ini berisi screenshot untuk dokumentasi sistem.

## Aturan Penamaan

- Gunakan prefix nomor untuk urutan pembacaan.
- Gunakan nama file deskriptif sesuai halaman.
- Contoh: 01-login-admin.png

## Tips Pengambilan Screenshot

- Gunakan data yang representatif dan tidak sensitif.
- Gunakan ukuran browser yang konsisten.
- Tampilkan area penting halaman, termasuk filter atau action utama.
